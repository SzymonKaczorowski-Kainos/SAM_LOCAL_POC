# SAM_LOCAL_POC

In order to run:

1. Go to:
```
/apis/functions/greeting
```

2. Run:
```
npm install
npm build:prod
```

3. Go to:
```
/apis
```

4. Build:
```
sam build
```

5. To run locally:
```
sam local start-api
```
